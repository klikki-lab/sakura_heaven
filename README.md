# SAKURA HEAVEN
  サクラをモチーフにしたリズムゲームです。Akashic Engine を使用しています。  

## 遊び方
  リズムに合わせてタイミングよくクリック。PCは'Z'キーでも可。
  目押しはとても難しいです。リズムにノッてクリックしよう！
